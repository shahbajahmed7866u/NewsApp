## Installation

To use NewsPedia, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/repository-name.git`
2. Install the dependencies: `npm install`

## Usage

To run NewsPedia, visit the website

<a href="https://newsapi.org/">newsapi.org</a>

Login with email and get your own api key

In React app create a file name ".env.local" and write:

```
REACT_APP_NEWS_API='XXXXXXXXXXX'
```

And then replace your api key with *XXXXXXXXXXX*


then execute the following command:

```
npm start
```

And use it the way you want.
